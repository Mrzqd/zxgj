## 5 试验方法

## 51 试件要求

双面暴露使用的人造板及其制品试件尺寸长度为 宽度为 试件数为块 采用不含甲醛的铝胶带封边 每块试件未封边长度为 当侧边面积小于暴露面总面积的 时 无须封边

单面暴露使用的人造板及其制品如木质地板 木质墙板 木质线条 木质门套 表面装饰材等只测试暴露面 试件尺寸长度为 $( 5 0 0 \pm 5 ) \ \mathrm { m m }$ 宽度为 $( 5 0 0 \pm 5 ) \ \mathrm { m m }$ ，样品尺寸不足时可拼接后制取，试件数量为4块，试件制作时采用不含甲醛的胶黏剂将2块试件背靠背粘起来，或者用不含甲醛的铝胶带将试件的背面密封 所有侧边应去除带榫舌的凸出部分后用不含甲醛的铝胶带密封

空心人造板 如空心刨花板 试件尺寸长度为 $( 5 0 0 \pm 5 ) \ \mathrm { \ m m } .$ <sub>,</sub>宽度为 $( 5 0 0 \pm 5 ) \ \mathrm { m m }$ <sub>,</sub>试件数量为块 试件有孔洞的边用不含甲醛的铝胶带封边 无孔洞边的未封边长度为 $( 7 5 0 \pm 5 ) \ \mathrm { m m }$ 。

集成材试件制作时不拼接 不封边 按全部表面积计算总暴露面积 总暴露面积为 $( 1 \pm 0 . 0 2 ) \textrm { m } ^ { 2 }$ 。

木质门扇试件包括 上梃或下梃和左右边梃 试件长度为门扇宽度 宽度根据测试面积截取1 $\mathrm { m ^ { 2 } }$ 仅正反面计入测试面积 其他部分应密封 不计入面积

小尺寸样品 如木质线条 木质门套等 可以采取拼接方式制作试件 试件采用不含甲醛的铝胶带封边 内部锯切拼缝需密封

## 52 试件预处理

试件在 相对湿度 条件下放置 空气置换率至少 次 试件之间距离至少25mm，使空气在所有试件表面上自由循环，空气中背景甲醛质量浓度不应超过$0 . 0 5 0 ~ \mathrm { m g / m ^ { 3 } }$ 。

注 使用空气净化装置来保持背景质量浓度不超过 $0 . 0 5 0 ~ \mathrm { m g / m ^ { 3 } }$

## 甲醛释放量测定

经5.1、5.2处理的试件，按照GB/T17657—2022中4.60.1\~4.60.3、4.60.5、4.60.6的规定进行测试 试件放入气候箱 内不需要取样 从 后开始 每 取样 次 每次取样时间间隔应超过如达到稳定状态 停止取样 如 内仍未达到稳定状态 则终止试验 甲醛释放量结果为最后次测定质量浓度的平均值

注：企业进行生产质量控制时，选择其他试验方法(见附录A)测定，并建立与本文件规定的 $1 ~ \mathrm { m ^ { 3 } }$ 气候箱法之间的相关性

## 6 判定规则

检验结果符合第 章限量标识规定的限量值时 判定为符合本文件规定的要求
